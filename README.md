# Installation
```
$ apt update && apt upgrade
$ apt install git python2
$ git clone https://github.com/d3m3nt3d7/FBI.git
$ cd fbi
```

# Setup
```
$ pip2 install -r requirements.txt
```
# Running
```
$ python2 fbi.py
```
